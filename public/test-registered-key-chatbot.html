🔑 RAG Chatbot - Registered Key Authentication Test

📋 How to use Registered Key Authentication:

  1. Register a user in the upload portal (http://localhost:3001/register)
  2. Copy the public key that's generated during registration
  3. Enter your username below to test the chatbot
  4. Initialize the chatbot with registered key authentication

🔑 Registered Key Authentication

📝 Configuration Example

Here's how to configure the chatbot with registered key authentication:

// Initialize chatbot with registered key authentication initRAGChatbot({ backendUrl: 'http://localhost:8001', auth: { useRegisteredKey: true, registeredUsername: 'your_username_here', uploadPortalUrl: 'http://localhost:3001', publicKeyEndpoint: '/api/auth/public-key' } });
