✨ Features

🔗 Ollama Integration

Seamlessly connects to your local Ollama LLM server for real AI responses. Supports any model you have installed.

🎨 Customizable Theme

Fully customizable colors and styling to match your website's design. Easy to integrate with any brand.

📄 Content Ingestion

Automatically captures page content for RAG (Retrieval-Augmented Generation) capabilities. Perfect for context-aware responses.

📍 Flexible Positioning

Position the chatbot anywhere on your page - fixed bottom-right corner or attach to any DOM element.

💬 Conversation Memory

Maintains conversation context across multiple messages for more coherent and helpful responses.

📱 Responsive Design

Works perfectly on desktop, tablet, and mobile devices with a clean, modern interface.

🔧 Integration Code

Add this simple code to any webpage to embed the chatbot:

<!-- Include the chatbot script --> <script src="chatbot.js"></script> <script> // Initialize with custom configuration initRAGChatbot({ model: 'mistral', theme: { primaryColor: '#667eea' }, welcomeMessage: 'Hello! I\'m your AI assistant. How can I help you today?', enableContentIngestion: true, ingestEndpoint: 'https://yourdomain.com/api/ingest' }); </script>

⚙️ Configuration Options

{ // Ollama model to use model: 'mistral', // Theme customization theme: { primaryColor: '#667eea' }, // Welcome message welcomeMessage: 'Hello! How can I help you?', // Content ingestion settings enableContentIngestion: true, ingestEndpoint: 'https://yourdomain.com/api/ingest', // Custom positioning (optional) selector: '#chat-container' }

🔍 Ollama Setup

📋 Prerequisites

  • Install Ollama: curl -fsSL https://ollama.ai/install.sh | sh
  • Start Ollama server: ollama serve
  • Pull a model: ollama pull mistral
