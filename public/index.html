✨ True RAG Features

🧠 Local Vector Embeddings

Uses sentence-transformers to generate embeddings locally in the browser. No external API calls needed for embedding generation.

🔍 Semantic Search

Performs cosine similarity search to find the most relevant content for each user query. Returns context with similarity scores.

📚 Vector Database

Local in-memory vector storage with efficient similarity search. Documents are indexed and searchable in real-time.

🎯 Context Injection

Automatically enhances prompts with relevant context from the vector database before sending to Ollama for more accurate responses.

📄 Content Ingestion

Automatically extracts, cleans, and embeds page content into the vector database for immediate RAG capabilities.

🤖 Ollama Integration

Seamlessly connects to local Ollama LLM with enhanced prompts that include retrieved context for better responses.

🔧 Integration Code

Add these scripts to any webpage to embed the True RAG chatbot:

<!-- Include RAG service and chatbot scripts --> <script src="rag-service.js"></script> <script> // Initialize with True RAG configuration initRAGChatbot({ model: 'mistral', theme: { primaryColor: '#667eea' }, welcomeMessage: 'Hello! I\'m your AI assistant with True RAG capabilities. I can search through this page\'s content to provide relevant answers!', enableRAG: true, enableContentIngestion: true, ragThreshold: 0.7, maxRAGResults: 3 }); </script>

⚙️ RAG Configuration Options

{ // Ollama model to use model: 'mistral', // Enable True RAG capabilities enableRAG: true, // RAG search parameters ragThreshold: 0.7, // Minimum similarity score (0-1) maxRAGResults: 3, // Max context sources to include // Content ingestion enableContentIngestion: true, // Theme customization theme: { primaryColor: '#667eea' }, // Welcome message welcomeMessage: 'Hello! How can I help you?' }

🔍 Ollama Setup

📋 Prerequisites

  • Install Ollama: curl -fsSL https://ollama.ai/install.sh | sh
  • Start Ollama server: ollama serve
  • Pull a model: ollama pull mistral

  • RAG technology and its benefits

  • Ollama integration and local LLMs
  • Chatbot widget features and capabilities

RAG (Retrieval-Augmented Generation) is a technique that combines the power of large language models with the ability to retrieve relevant information from external knowledge sources. This approach enhances the model's responses by providing it with specific, up-to-date information that may not be part of its training data.

Vector embeddings are numerical representations of text that capture semantic meaning. They allow us to perform similarity searches and find related content even when the exact words don't match. This is crucial for effective information retrieval in RAG systems.

Ollama is an open-source framework that allows you to run large language models locally on your machine. It provides a simple API for interacting with various models like Mistral, Llama, and others without requiring cloud services or internet connectivity.
