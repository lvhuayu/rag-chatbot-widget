🤖 RAG Chatbot 最小化功能演示

✨ 新增功能

  • 聊天窗口可以最小化为圆形图标
  • 点击最小化按钮隐藏聊天界面
  • 点击圆形图标重新展开聊天窗口
  • 平滑的动画过渡效果
  • 悬停效果增强用户体验
  • 支持自定义容器和固定定位

📋 使用说明

  1. 聊天机器人会在页面右下角显示
  2. 点击聊天窗口右上角的最小化按钮（📊图标）
  3. 聊天窗口会平滑缩小并隐藏
  4. 此时会显示一个圆形机器人图标（🤖）
  5. 点击圆形图标可以重新展开聊天窗口
  6. 所有动画都有平滑的过渡效果

🎯 测试区域

聊天机器人将在页面右下角显示。尝试点击最小化按钮来测试功能！

🔧 技术特性

  • 使用 CSS transform 和 opacity 实现平滑动画
  • 通过 postMessage API 实现 iframe 与父窗口通信
  • 响应式设计，支持不同屏幕尺寸
  • 优雅的悬停效果和视觉反馈
  • 保持原有的所有 RAG 和认证功能
