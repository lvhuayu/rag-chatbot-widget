🔑 Multi-Tenant RAG Chatbot

Each user gets their own isolated knowledge base. No data mixing between different users.

🎮 Interactive Demo

Ready to initialize chatbot. Enter a Site ID and click "Initialize Chatbot".

1Data Attribute Method

Add the siteId directly to the script tag using data attributes.
<script src="chatbot.js" data-site-id="company_abc"></script> <script> initRAGChatbot({ backendUrl: "http://localhost:8001" }); </script>

2Configuration Object Method

Pass the siteId in the configuration object when initializing.
<script src="chatbot.js"></script> <script> initRAGChatbot({ siteId: "site123", backendUrl: "http://localhost:8001" }); </script>

3Global Config Method

Set the siteId in a global configuration object before loading the script.
<script> window.__RAG_CHATBOT_CONFIG__ = { siteId: "enterprise_xyz" }; </script> <script src="chatbot.js"></script> <script> initRAGChatbot({ backendUrl: "http://localhost:8001" }); </script>

4Element Attribute Method

Add the siteId to any element on the page using data attributes.
<div data-rag-site-id="startup_123"></div> <script src="chatbot.js"></script> <script> initRAGChatbot({ backendUrl: "http://localhost:8001" }); </script>
